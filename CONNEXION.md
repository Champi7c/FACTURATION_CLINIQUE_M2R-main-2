# Coordonnées pour se connecter

Après avoir créé les utilisateurs (voir ci‑dessous), utilisez ces identifiants sur l’écran de connexion.

## Utilisateurs par défaut

| Rôle         | Nom d'utilisateur | Mot de passe   |
|-------------|-------------------|----------------|
| **Super Admin** | `administrateur`   | `FACTURATION2025` |
| **Manager**     | `manager`          | `FACTURATION2025` |

## Créer les utilisateurs (première fois)

Depuis le dossier **backend** :

```bash
python manage.py create_users
```

Ensuite, ouvrez l’application (frontend) et connectez-vous avec un des comptes ci‑dessus.
